# 个人博客项目需求文档 (X风格)

## 1. 项目概述

- **项目名称**: X-Style Blog
- **项目类型**: 个人博客网站
- **技术栈**: Next.js + Tailwind CSS + TypeScript
- **核心风格**: 模仿X (Twitter) 的UI设计
- **目标用户**: 访客、读者

---

## 2. 功能模块规划

### 2.1 首页 (Home)

**描述**: 类似于X的时间线，展示最新内容

**包含内容**:
- 顶部导航栏（Logo、搜索、头像）
- "为你推荐" / "关注" 标签切换
- 内容流（混合展示长文章和短动态）
- 侧边栏：热门话题/推荐内容

**内容卡片**:
- 长文章卡片：标题、摘要、封面图、阅读量、评论数、发布时间
- 短动态卡片：内容、发布时间、点赞/评论数

### 2.2 探索 (Explore)

**描述**: 发现内容、搜索、分类浏览

**包含内容**:
- 搜索栏（支持关键词搜索文章和动态）
- 热门文章列表
- 分类标签云（技术、生活、思考等）
- 近期热门短动态

### 2.3 文章详情页

**描述**: 阅读长文章的详细页面

**包含功能**:
- 文章标题、作者信息、发布时间、阅读量
- 文章正文（支持Markdown渲染）
- 目录导航（可选）
- 点赞按钮
- 评论区域
- 相关文章推荐

### 2.4 短动态页

**描述**: 类似于X的推文流

**包含功能**:
- 发布框（可以发布短动态）
- 短动态列表
- 点赞、评论功能
- 分页/无限滚动

### 2.5 个人资料页 (Profile)

**描述**: 博主个人介绍和内容归档

**包含内容**:
- 头像、用户名、简介、背景图
- 关注数/粉丝数（可省略或展示为虚拟数字）
- 文章Tab
- 短动态Tab
- 关于Tab（详细个人简介）

### 2.6 评论系统

**描述**: 简单的评论功能

**包含功能**:
- 评论输入框
- 评论列表（头像、用户名、内容、时间）
- 简单防刷（可选）

---

## 3. 非功能需求

### 3.1 性能要求
- 首屏加载时间 < 2秒
- 支持SSR/SSG优化SEO

### 3.2 响应式设计
- 桌面端：类似X的三栏布局
- 平板端：两栏布局
- 移动端：单栏布局，底部导航

### 3.3 SEO
- 静态页面生成
- Meta标签优化
- sitemap支持

## 5. 数据结构 (初步)

### 文章 (Post)
```
- id: string
- title: string
- slug: string
- content: string (markdown)
- excerpt: string
- coverImage?: string
- category: string
- tags: string[]
- viewCount: number
- createdAt: Date
- updatedAt: Date
```

### 短动态 (MicroPost)
```
- id: string
- content: string (max 280 chars)
- createdAt: Date
- likes: number
```

### 评论 (Comment)
```
- id: string
- postId?: string
- microPostId?: string
- author: string
- content: string
- createdAt: Date
```
